## 배운 핵심 개념
- **venv** — 프로젝트별 독립 Python 환경
- **`-m` 플래그** — 프로젝트 루트 기준 모듈 실행 (슬래시→점, .py 제거)
- **`if __name__ == "__main__"`** — 직접 실행할 때만 동작, import 시엔 무시
- **타입 힌트** (`list[str]`, `dict[str, pd.DataFrame]`) — 실행엔 영향 없는 설명서 역할
- **`Path.parents[N]`** — 현재 파일 기준 상위 N번째 폴더 (parents[0]=src/data/, parents[1]=src/, parents[2]=프로젝트 루트)
- **`mkdir(parents=True, exist_ok=True)`** — 중간 폴더 포함 생성, 이미 있어도 에러 없음
- **`yf.Ticker().history()`** — yfinance 새 방식. `download()` 보다 컬럼 구조 단순
- **`import ... as`** — import할 때 이름 바꾸기 (충돌 방지)
