```

────────────────────         ────────────────────────
```

### Streamlit 대시보드 구성 패턴

```python
st.set_page_config(layout="wide")

# KPI 카드
c1, c2, c3, c4 = st.columns(4)
c1.metric("제목", "값", "변화량")

# 탭
tab1, tab2 = st.tabs(["탭1", "탭2"])
with tab1:
    st.plotly_chart(fig, use_container_width=True)
```

- `st.metric()` = KPI 카드 (제목 / 값 / 델타)
- `st.tabs()` = 탭 구조
- `st.dataframe(df.style.map(fn))` = 셀 색상 입히기
